PostgreSQL 13.4, compiled by Visual C++ build 1914, 64-bit

1. Para la creación de la base de datos es necesario asignar 
el tablespace si se tiene uno específico en el archivo
db_create.sql

2. Arrastrar db_create.sql dentro del querytool de pgadmin y
ejecutar

3. Arrastrar db_model1_table_create.sql dentro del querytool
de pgadmin y ejecutar

4. Dentro de la carpeta data, arrastrar los archivos values
dentro del query tool de pgadmin y ejecutar según el número 
de orden que está nombrado

Adicional. Si requiere reiniciar los valores SERIAL dentro
de la base de datos, arrastre data_model1_tables_resetSerial.sql
dentro del querytool de pgadmin y ejecutar
